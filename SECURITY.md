# Security Policy

If you find a security issue in `AgentInbox`, please do not open a public issue
with exploit details.

Instead, report it privately to the maintainers through GitHub Security
Advisories:

- https://github.com/holon-run/agentinbox/security/advisories/new

Include:

- affected version
- impact summary
- reproduction steps
- any suggested mitigation

We will triage, reproduce, and coordinate a fix before public disclosure.
